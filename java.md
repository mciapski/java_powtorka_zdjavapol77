# Quiz wiadomości o języku Java

1. Gdy klasa nie ma zaimplementowanego konstruktora, automatycznie tworzony jest w niej _____.
2. Wszystkie metody w interfejsie są z automatu _____.
3. Gdy metoda rzuca wyjątek typu Checked Exception, przy jej użyciu należy albo _____ albo _____.
4. Klasa może zaimplementować _____ interfejs/interfejsów oraz rozszerzyć _____ klasę/klas (uzupełnij luki odpowiednią liczbą lub słowem "nieograniczoną liczbę").
5. Jeżeli mamy w klasie wiele metod o tej samej nazwie przyjmujących różne argumenty to nazywa się to _____ (podaj nazwę zabiegu po polsku lub angielsku).
